##THEORY  
**What are file permissions in Linux?**  
File permissions define access levels:  
- r = read  
- w = write  
- x = execute  
They are assigned to: owner, group, and others.  

**What is the difference between root and normal user?**  
Root user has full system access and control.  
Normal user has limited permissions and restricted access.
